# Log Copied

Does what it says on the label, `console.log()`s what you just copied

## How to use

Just copy something and hit <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>L</kbd> 

(<kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>L</kbd> on mac)
